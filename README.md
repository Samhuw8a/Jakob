# Jakobsstab
Ein Mathe Projekt zum Jakobsstab

# Dependencies
- cx_Freeze
- python3
- pyyaml
- tkinter
- patchelf

# Instalation
> Selber bauen
> ---
> - `git clone https://github.com/Samhuw8a/Jakob.git`
> - `cd Jakob`
> - `sudo make compile`
